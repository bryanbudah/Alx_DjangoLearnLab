# Django Blog Project - Setup Instructions

This document explains how to set up the `django_blog` project locally for development and testing.

---

## 1. Prerequisites

- Python 3.11+ installed
- pip (Python package manager)
- Virtual environment tool (`venv` or `virtualenv`)
- PostgreSQL (optional: for production setup) or SQLite (default dev)
- Git (optional for cloning repo)
